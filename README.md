# 项目名及简介
* 基于appium+python3封装的自动化测试框架

# 介绍
* unittest参数化
* PageObject分层管理
* 本地化多机日志管理
* 用例编写基于yaml配置多关键字驱动
* 支持多设备andoird并行
* 支持失败重连
* 自动生成excel测试报告



## 命令运行

```
python runner.py
```
